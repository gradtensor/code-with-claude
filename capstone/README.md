# Capstone: AI Document Analyst

A working agent that takes a corpus of documents (PDFs, emails, transcripts) and produces structured analysis, summaries, and grounded answers. The capstone grows one capability per day across the 8-day series:

- v1: prompt-only baseline (Day 1)
- v2: context and MCP integration (Day 2)
- v3: tool use and agent loop (Day 3)
- v4: workflow patterns (Day 4)
- v5: analysis and visualisation (Day 5)
- v6: evaluation pipeline (Day 6)
- v7: orchestration and tracing (Day 7)
- v8: full showcase (Day 8)

Each version lives in its own subfolder and is independently runnable.
