# Day 1: Advanced prompt engineering with the GCAO framework

Prompt engineering is the foundation of every later capability. This day introduces the GCAO framework (Goal, Context, Audience, Output) as a repeatable structure for production prompts. You will see how each element shapes the response, when to use system prompts versus user messages, and how small changes in instruction wording move quality and cost. The capstone v1 ships as a prompt-only document analyst built on these patterns.
