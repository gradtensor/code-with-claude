│ ## 🧵 Union Budget 2026-27: Impact on the Textile Industry                   │
│                                                                              │
│ ### 1. 🏭 Integrated Programme for the Textile Sector (Para 22)              │
│ The Budget announces a comprehensive **Integrated Programme** specifically   │
│ for the **labour-intensive Textile Sector**, comprising **five sub-parts**:  │
│                                                                              │
│ | Sub-scheme | Purpose |                                                     │
│ |---|---|                                                                    │
│ | **(a) National Fibre Scheme** | Self-reliance in natural fibres (silk,     │
│ wool, jute), man-made fibres, and new-age fibres |                           │
│ | **(b) Textile Expansion and Employment Scheme** | Modernise traditional    │
│ clusters with capital support for machinery, technology upgradation, and     │
│ common testing & certification centres |                                     │
│ | **(c) National Handloom and Handicraft Programme** | Integrate existing    │
│ schemes; targeted support for weavers and artisans |                         │
│ | **(d) Tex-Eco Initiative** | Promote globally competitive and              │
│ **sustainable** textiles and apparel |                                       │
│ | **(e) Samarth 2.0** | Modernise and upgrade the textile **skilling         │
│ ecosystem** through collaboration with industry and academia |               │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 2. 🏗️ Mega Textile Parks (Para 23)                                       │
│ The Budget proposes to set up **Mega Textile Parks** in **challenge mode**,  │
│ with a focus on bringing **value addition to technical textiles** — a        │
│ high-growth segment globally.                                                │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 3. 🪡 Mahatma Gandhi Gram Swaraj Initiative (Para 24)                    │
│ A dedicated initiative to **strengthen khadi, handloom and handicrafts**,    │
│ with objectives including:                                                   │
│ - Global market linkage and branding                                         │
│ - Streamlined support for training, skilling, and quality of                 │
│ process/production                                                           │
│ - Benefits for **weavers, village industries, the One District–One Product   │
│ initiative**, and rural youth                                                │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 4. 📦 Export Promotion Measures for Textile Exporters                    │
│                                                                              │
│ #### Extended Export Timeline (Para 149 / Annexure D.2)                      │
│ - The time period for the **export of final products** using duty-free       │
│ imported inputs is being **extended from 6 months to 12 months** for         │
│ exporters of **leather or textile garments**, leather/synthetic footwear,    │
│ and other leather products. This gives textile exporters significantly more  │
│ breathing room in their production and export cycles.                        │
│                                                                              │
│ #### Duty-Free Inputs Eligibility Extended (Para 148 / Annexure D.3)         │
│ - The benefit of **duty-free imports of specified inputs**, previously       │
│ available for leather/synthetic footwear exports, is now extended to **Shoe  │
│ Uppers** as well — benefiting businesses at the intersection of textile and  │
│ footwear.                                                                    │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 5. 🧶 Fibre & Sustainability Focus                                       │
│ The **Tex-Eco Initiative** signals a clear policy push toward **eco-friendly │
│ and sustainable textiles**, aligning India's textile sector with global      │
│ green standards — crucial for export competitiveness, especially in European │
│ and North American markets.                                                  │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 6. 🔧 Indirect Tax / Customs — Broader Enablers                          │
│ While no specific customs duty cuts were announced for textile raw materials │
│ in this Budget, the **overall trade facilitation measures** benefit textile  │
│ exporters:                                                                   │
│ - **Removal of ₹10 lakh cap on courier exports** (Para 175) helps textile    │
│ artisans and small businesses access global e-commerce markets.              │
│ - **Advance Ruling validity extended from 3 to 5 years** (Para 165) provides │
│ certainty for textile import/export planning.                                │
│ - **AEO duty deferral extended from 15 to 30 days** (Para 163) improves      │
│ liquidity for large textile importers.                                       │
