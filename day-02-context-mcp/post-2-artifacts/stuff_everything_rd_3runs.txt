│ ## R&D-Related Allocations in Budget 2026-27                                 │
│                                                                              │
│ Based on the budget speech, here are all the R&D-related mentions across     │
│ sectors. Note that **explicit standalone R&D budget figures are limited**;   │
│ most are embedded within broader scheme outlays.                             │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 1. Biopharma SHAKTI                                                      │
│ - **Outlay: ₹10,000 crore over 5 years**                                     │
│ - Covers building the ecosystem for domestic production of biologics and     │
│ biosimilars, including 3 new NIPERs and upgrading 7 existing ones — which    │
│ inherently include research and education infrastructure.                    │
│ - Also creates a network of over 1,000 accredited India Clinical Trials      │
│ sites, which is research-infrastructure spending.                            │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 2. India Semiconductor Mission (ISM) 2.0                                 │
│ - **No specific outlay stated** in the speech.                               │
│ - Explicitly mentions "industry-led research and training centres to develop │
│ technology and skilled workforce."                                           │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 3. Carbon Capture Utilization and Storage (CCUS)                         │
│ - **Outlay: ₹20,000 crore over 5 years**                                     │
│ - Described as achieving "higher readiness levels" in technology across      │
│ power, steel, cement, refineries and chemicals — this is fundamentally an    │
│ R&D and technology development programme.                                    │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 4. Rare Earth Permanent Magnets / Rare Earth Corridors                   │
│ - **No specific outlay stated.**                                             │
│ - Proposes dedicated Rare Earth Corridors in Odisha, Kerala, Andhra Pradesh  │
│ and Tamil Nadu to promote "mining, processing, **research** and              │
│ manufacturing."                                                              │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 5. Sports — Khelo India Mission                                          │
│ - **No specific outlay stated.**                                             │
│ - Explicitly includes "integration of sports science and technology" as one  │
│ of the Mission's pillars.                                                    │
│ - The proposed sports goods initiative also mentions "research and           │
│ innovation in equipment design as well as material sciences."                │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 6. AYUSH                                                                 │
│ - **No specific outlay stated.**                                             │
│ - Proposes to upgrade the **WHO Global Traditional Medicine Centre in        │
│ Jamnagar** to "bolster **evidence-based research**, training and awareness   │
│ for traditional medicine."                                                   │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 7. Telescope and Astronomy Infrastructure                                │
│ - **No specific outlay stated.**                                             │
│ - Proposes setting up or upgrading 4 facilities: National Large Solar        │
│ Telescope, National Large Optical-infrared Telescope, Himalayan Chandra      │
│ Telescope, and COSMOS-2 Planetarium — all fundamentally research             │
│ infrastructure.                                                              │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 8. Existing Technology Missions (Referenced, Not New Allocations)        │
│ - The speech references previously established platforms — **AI Mission,     │
│ National Quantum Mission, Anusandhan National Research Fund, and Research,   │
│ Development and Innovation Fund** — as part of the government's technology   │
│ support ecosystem, but **no new allocations for these are announced** in     │
│ this budget speech.                                                          │
│                                                                              │
│ ---                                                                          │
│                                                                              │
│ ### 9. Medical Value Tourism Hubs                                            │
│ - **No specific outlay stated per hub.**                                     │
│ - The 5 Regional Medical Hubs will combine "medical, **educational and       │
│ research** facilities."                                                      │
│                                                                              │
│ ---                                                                          │
│                                                                              │

│ ### Tax Incentives Supporting R&D/Innovation                                 │
│                                                                              │
│ - **Tax holiday till 2047** for foreign companies providing cloud services   │
│ using Indian data centres                                                    │
│ - **5-year income tax exemption** for non-residents providing capital        │
│ goods/equipment/tooling to toll manufacturers in bonded zones (aimed at      │
│ electronics manufacturing R&D)                                               │
│ - **Safe harbour for IT services** threshold raised from ₹300 crore to       │
│ **₹2,000 crore**, covering contract R&D services relating to software        │
│ development                                                                  │
│ - **Customs duty exemption** on capital goods for **critical minerals        │
│ processing**                                                                 │
│                                                                              │
│ ---                                                                          │
│                                                                              │

│ for these in this budget speech.                                             │
